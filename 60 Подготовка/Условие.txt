Подготовка
Вася решил как следует подготовиться ко второму туру олимпиады по информатике. Для этого он решил устроить самому себе n небольших олимпиад, в каждой по m задач. Для того, чтобы составить задания для этих олимпиад, Васе необходимы сборники олимпиадных задач. Эти сборники есть в библиотеке. Известно, что их там ровно k. Причем i-ый сборник содержит ровно ai задач. Васе не хочется все время бегать в библиотеку, он хочет за один раз взять нужное количество сборников, чтобы составить для себя все n олимпиад. Каково минимальное количество сборников, которое ему нужно для этого взять в библиотеке?
Входные данные
Первая строка содержит три натуральных числа k, m, n, где k - количество сборников в библиотеке, m - количество задач в одной олимпиаде, n - количество олимпиад (1 ≤ k ≤ 100 000, 1 ≤ m, n ≤ 10 000). Вторая строка содержит k натуральных чисел a1, ..., ak. Известно что ai (1 ≤ ai ≤ 109) - это количество задач в i-ом сборнике.
Выходные данные
Вывести минимальное количество сборников, которое Вася должен взять в библиотеке. Считайте, что в библиотеке достаточно сборников, чтобы Вася мог составить олимпиадные задания.
