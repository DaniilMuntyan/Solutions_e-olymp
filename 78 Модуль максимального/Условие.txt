Модуль максимального
Задана последовательность действительных чисел. Вычислите их модули. Определите максимальное значение среди этих модулей.
Входные данные
В первой строке содержится количество элементов n (n ≤ 100) в последовательности. В следующей строке заданы n действительных чисел - элементы последовательности, значения которых не превышают по модулю 100.
Выходные данные
Вывести максимальное значение среди модулей заданных чисел с 2 десятичными знаками.
