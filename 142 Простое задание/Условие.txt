Простое задание
Для заданного целого числа n вычислите сумму всех целых чисел от 1 до n (включая 1 и n).
Входные данные
В каждой строке содержится единственное целое число n (|n| ≤ 10000). Последняя строка содержит 0 и не обрабатываться.
Выходные данные
Для каждого входного числа выведите ответ в отдельной строке.
