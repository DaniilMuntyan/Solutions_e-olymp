Сумма элементов
Задана последовательность из n действительных чисел. Найти сумму всех ее элементов.
Входные данные
В первой строке записано число n (n ≤ 100). В следующей строке записано n действительных чисел, каждое из которых не превосходит по модулю 100.
Выходные данные
Вывести сумму всех элементов последовательности с одним десятичным знаком.
