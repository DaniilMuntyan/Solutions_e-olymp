Подсчет количества ребер
Ориентированный граф задан матрицей смежности. Найдите количество ребер в графе.
Входные данные
Первая строка содержит количество вершин в графе n (1 ≤ n ≤ 100). Далее следуют n строк по n чисел, каждое из которых равно 0 или 1 - матрица смежности графа.
Выходные данные
Выведите количество ребер графа.
