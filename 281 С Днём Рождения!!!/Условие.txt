С Днём Рождения!!!
У кошки Сони день рождения в конце марта. Как раз тогда, когда все коты и их подружки любят гулять. У Сони есть k друзей, и все они хотят поздравить её. Для этого Соня решила собрать их всех в одном месте, чтобы потом отправиться куда-нибудь погулять. Соня очень любит своих друзей, поэтому, хочет выбрать место встречи так, чтобы сумма всех расстояний, пройденных её друзьями до этого места, была минимальной.
Так как, Соня увлекается олимпиадным программированием, она решила усложнить задачу, сделав пространство n - мерным. Расстоянием между точкой (x1, x2, ..., xn) и точкой (y1, y2, …, yn) будем считать сумму (|x1 - y1| + |x2 - y2| + ... + |xn - yn|).
Входные данные
В первой строке даются два числа: k (1 ≤ k ≤ 150000) и n (1 ≤ n ≤ 10). Каждая из следующих k строк содержит n целых чисел – координаты точки расположения текущего друга Сони. Координаты не превосходят по модулю 50000.
Выходные данные
Выведите координаты точки, оптимальной для встречи. Следуйте формату вывода, показанному в примере. Если таких точек несколько, выведите ту, у которой первая координата меньше. Если их первые координаты равны, сравните вторые, и так далее. Затем, выведите одно число - суммарное расстояние, пройденное друзьями Сони.
