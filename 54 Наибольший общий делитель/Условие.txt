Наибольший общий делитель
Даны n натуральных чисел. Напишите программу, вычисляющую наибольший общий делитель этих чисел.
Входные данные
В первой строке находится натуральное число n (n ≤ 1000) - количество чисел. Далее содержатся n натуральных чисел, каждое из которых не превосходит 2 * 109.
Выходные данные
Выведите единственное число - наибольший общий делитель данных чисел.
