Фибоначчи
Числа Фибоначчи - это последовательность чисел F(n), которая задаётся формулой:
F(0) = 1, F(1) = 1, F(n) = F(n - 1) + F(n - 2)
По заданному числу n вывести n-ое число Фибоначчи.
Входные данные
Неотрицательное число n (n ≤ 45) - номер числа Фибоначчи, которое нужно вывести.
Выходные данные
Вывести n-ое число Фибоначчи.
