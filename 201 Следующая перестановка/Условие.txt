Следующая перестановка
Найдите следующую перестановку. Тождественная перестановка является следующей для обратной.
Входные данные
В первой строке записано количество элементов n (1 ≤ n ≤ 105) в перестановке. Во второй строке записана перестановка из n чисел.
Выходные данные
Вывести n чисел - искомую перестановку.
