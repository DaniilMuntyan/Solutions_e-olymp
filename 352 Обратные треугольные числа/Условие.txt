Обратные треугольные числа
Треугольным называется число, которое может быть представлено множеством точек, упакованных в равносторонний треугольник с n точками на стороне. Далее приведены примеры треугольников с соответствующими треугольными числами:

По заданному числу определите, является ли оно треугольным. Если ответ положительный, то определите количество точек на его стороне.
Пример: Если k = 10, то оно является треугольным с 4 точками на стороне, так как 10 = 4 + 3 + 2 + 1. Если k = 11, то оно не является треугольным.
Входные данные
Состоит из нескольких строк. Каждая строка содержит целое число n (0 < n < 109) без ведущих нулей. Последняя строка содержит -1 и не обрабатывается.
Выходные данные
Определить, является ли значение n треугольным числом. Если ответ положительный, то вывести в строке количество точек на стороне. Если ответ отрицательный, вывести строку "bad".
