Степени вершин
Простой неориентированный граф задан матрицей смежности. Найдите степени всех вершин графа.
Входные данные
В первой строке задано количество вершин графа n (1 ≤ n ≤ 100). Затем идут n строк по n элементов в каждой - описание матрицы смежности.
Выходные данные
Выведите n чисел - степени всех вершин.
