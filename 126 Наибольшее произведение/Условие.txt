Наибольшее произведение
Для каждого положительного числа x определим функцию P(x), равную произведению цифр в десятичной записи числа x. По заданному N вычислить максимальное значение P(x) при x ≤ N.
Входные данные
Во входном файле содержится одно целое число N (1 ≤ N≤ 2·109).
Выходные данные
Выведите максимальное значение функции P(x) на промежутке от 1 доN.
