Кафе "Хоботания"
В городе Е. открылось новое кафе "Хоботания", расчитанное на слоников. Все клиенты приходят в кафе в момент времени 0, а владелец выбирает в каком порядке их обслуживать. При этом каждую секунду обслуживается один слоник (первый обслуживается в момент времени 0).
Владельцу известно, что если слоник i будет обслужен в момент времени t, то он заплатит tipsi - t чаевых. Если число tipsi - t отрицательно, то он ничего не платит.
Помогите владельцу найти такой порядок обслуживания слоников, который принесёт ему максимальную прибыль.
Входные данные
Первая строка содержит количество слоников n (0 ≤ n ≤ 100), пришедших в кафе. Следующая строка содержит n чисел tips1, tips2, ..., tipsn (0 ≤ tipsi ≤ 105).
Выходные данные
Выведите максимальную прибыль владельца кафе.
