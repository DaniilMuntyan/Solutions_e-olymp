Сортировка времени
Отсортируйте время согласно заданному критерию.
Входные данные
Сначала задано число n (1 ≤ n ≤ 100), а затем n моментов времени. Каждый момент времени задается 3 целыми числами - часы (от 0 до 23), минуты (от 0 до 60) и секунды (от 0 до 60).
Выходные данные
Выведите моменты времени, упорядоченные в порядке неубывания (момент времени также выводится в виде трех чисел, ведущие нули выводить не нужно).
