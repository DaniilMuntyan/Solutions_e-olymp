Странности
Некоторые числа нечетные. Например, число 3 нечетное, так как не делится на два. Числа, которые делятся на два, нечетными не будут, они называются четными. Более точно, если число n можно представить в виде n = 2 ∗ k для некоторого целого k, то n четно. Например, 6 = 2 ∗ 3 четно.
Некоторые люди путаются, является ли число четным или нечетным. Чтобы разобраться, Вы можете задать запрос интернет поисковику "является ли число четным или нечетным?" (Не выполняйте поиск! Решите задачу!)
Напишите программу для таких людей.
Входные данные
Начинается строкой с количеством входных данных n (1 ≤ n ≤ 20). Каждая из следующих n строк содержит тест, содержащий одно целое число x (-10 ≤ x ≤ 10).
Выходные данные
Для каждого x выведите или 'x is odd', или 'x is even' в зависимости от того, является ли x нечетным или четным.
