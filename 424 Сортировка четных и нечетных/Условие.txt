Сортировка четных и нечетных
Задана последовательность целых чисел. Отсортируйте заданную последовательность так, чтобы сначала были расположены нечетные числа в возрастающем порядке, а потом четные числа в убывающем порядке.
Входные данные
Первая строка содержит количество чисел n (1 ≤ n ≤ 1000). Вторая строка содержит n чисел, не превосходящих по модулю 2 * 109.
Выходные данные
В одной строке выведите последовательность чисел, упорядоченную согласно условию задачи.
