Контакты
Мой мобильный настроен так, что каждый номер телефона после любого соединения автоматически записывается в книгу контактов, при условии, что такого номера там ещё нет. Найти количество записей в книге контактов, если известен список номеров телефонов всех N мобильных соединений.
Входные данные
Число N в первой строке ( 1< ≤ N ≤ 105 ). Во второй N номеров телефонов, каждый из которых девятизначное натуральное число.
Выходные данные
Количество записей в книге контактов.
