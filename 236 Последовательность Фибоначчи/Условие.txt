Последовательность Фибоначчи
Последовательность Фибоначчи задается следующим образом:
a0 = 0
a1 = 1
ak = ak-1 + ak-2
Для заданного n найти значение n-го элемента an последовательности Фибоначчи.
Входные данные
Одно натуральное число n (1 ≤ n ≤ 40).
Выходные данные
Вывести n-ый элемент последовательности Фибоначчи.
