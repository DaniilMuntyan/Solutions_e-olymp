Запрос Суммы на Отрезке
Имеется список L, который содержит n целых чисел. Найдите сумму на отрезке (RSQ) между индексами i и j включительно, то есть
RSQ(i, j) = Li + Li+1 + Li+2 + ... + Lj
Входные данные
Начинается с количества тестов t (1 ≤ t ≤ 5). Каждый тест начинается с пустой строки, за которой идет строка с двумя целыми числами n и q (1 ≤ n, q ≤ 105). Следующая строка содержит n неотрицательных целых чисел до 109. Далее следуют q строк, каждая из которых содержит два целых числа i и j (0 ≤ i, j < 10000).
Выходные данные
Для каждого запроса вывести в отдельной строке значение RSQ(i, j). Разделяйте ответы на соседние тесты пустой строкой.
