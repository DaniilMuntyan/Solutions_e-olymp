Олимпиада
На олимпиаду по информатике прибыли n команд по Ai (1 ≤ i ≤ n) участников в каждой. Для проведения соревнований приготовили классы с одинаковым количеством m компьютеров в каждом. Какое минимальное количество классов необходимо задействовать при условии, что в каждом классе будут представители только разных команд. То есть ни в каком классе не должно находится более одного участника из одной команды.
Входные данные
В первой строке заданы числа n и m. Во второй строке находятся n чисел Ai (1 ≤ i ≤ n). Числовые значения целые, неотрицательные и не превышают 100.
Выходные данные
Вывести одно число - необходимое количество классов.
